# Attio

Sleek , accurate and modern House price predictor

# ⛏️ Installation

To follow along the tutorial. Change to the starter branch after cloning the repo.
Then install the npm packages & run

```bash
npm install && npm run dev
```

## 🔓 Stack

- Next14
- Typescript
- Shadcn
- Tailwind
